# 贡献者

感谢每一位为本项目做出贡献的人！无论是代码、设计、插画、问题反馈还是内容建议，你们的帮助都让这个教程变得更好。

## 贡献类型

| 图标 | 类型 | 说明 |
|:----:|------|------|
| 🎨 | 界面设计 | UI/UX 设计、视觉优化 |
| 🖼️ | 插画/配图 | 教程插图、架构图、流程图 |
| 🐛 | 问题反馈 | 通过 Issue、微信、QQ、邮件等渠道反馈错误 |
| 📝 | 内容建议 | 提供内容改进建议、学习路径优化 |
| 🔍 | 审阅 | 内容审阅、技术校对 |
| 🌍 | 翻译 | 英文翻译或其他语言翻译 |
| 💡 | 示例代码 | 提供代码示例或改进建议 |
| 📢 | 推广宣传 | 帮助传播项目 |

## 如何成为贡献者

我们欢迎各种形式的贡献！包括但不限于：

- **提交 Issue**：发现错误或不准确之处
- **内容建议**：希望增加的主题或改进建议
- **界面设计**：文档站 UI/UX 优化方案
- **插画配图**：为教程绘制示意图、流程图
- **翻译**：帮助翻译教程内容
- **代码示例**：提供更好的代码示例
